# myapp
En lenguaje dart flutter
Crea una app en Flutter para una clínica dental llamada 'Great Smile Clinic'. Todo el código debe ir en un solo archivo (para DartPad) y seguir estas reglas exactas:

Navegación Pro: Usa Rutas Nombradas (/, /tratamientos, /resenas, /cita, /especialistas). Nada de navegación simple, solo rutas por nombre.

Inicio (Home) Minimalista: >    * Centra todo y usa un fondo color cian pastel.

Pon una imagen redonda de una muela (usa CircleAvatar con sombra).

Abajo pon mi nombre: 'Gioser Fisher' (que se vea grande y en negrita).

Agrega una descripción corta y elegante de la clínica (máximo 2 líneas).

Drawer (Barra lateral):

Pon un encabezado (DrawerHeader) con el logo redondo de la clínica, el nombre, la dirección, teléfono y correo. Que se vea ordenado y con iconos.

Agrega 4 opciones con ListTile: Tratamientos, Reseñas, Agendar Cita y Especialistas. Cada una debe llevar a su página correspondiente usando el nombre de la ruta.

Páginas de la Barra (Interiores): >    * No les pongas texto ni nada raro. Solo pon una imagen centrada (de 200x200 o 250x250) traída de GitHub o una URL de red.

Ponles bordes redondeados y una sombra suave para que se vea 'caro'.

Abajo de la imagen pon un único botón de 'Volver' que regrese al inicio.

Diseño y Colores: Usa una paleta de colores Teal, Cian y Blanco. Nada de diseños básicos; usa sombras (BoxShadow), bordes redondeados (ClipRRect) y buen espaciado para que el diseño se vea profesional y 'bonito'."
